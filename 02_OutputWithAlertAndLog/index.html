What's the difference between alert() and console.log()?

The difference between alert and console log is that alert brings up a pop up type box on the page and console.log() stores things on the browsers console.

What's one situation where you might use an `alert()` to tell the user something?

One situation where you may want to use an 'alert()' would be to notify them when they are about to delete or change something on a page. Something like an "Are you sure you want to do this? Y/N" situation.

What's one way that `alert()` might get obnoxious?

An 'alert()' might get a=obnoxious if they're over used or for example, pop ups and things.
