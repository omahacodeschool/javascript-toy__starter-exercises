Why don't you need to use the var keyword every time you use a variable (right now, we only use it when we create the variable)? What does var do?

The reason we do not need to use the var keyword each time we use a variable is because the var keyword is used only to indicate to the javascript interpretor (or whatever it's called) that we're about to declare a variable for the first time. After the variable has been declared, we can then use that variable's name without the var keyword from that point forward (even when redefining the variable).
