Basic Math

Question: What happens when you try to parse the integer out of a string like "I am 9 years old"? How about "9 is my age"? Other combinations of numbers and letters? What needs to be true about the incoming string for it to be parsed correctly?

When we try to parse the integer out of a string we get a 'NaN + NaN + NaN = NaN' result, except in the situation when a user inputs an integer first, for example: 9 is my age will be recorded as var = 9. Therefore, for a string to be parsed correctly into an integer, the first character of the string must be an integer.
