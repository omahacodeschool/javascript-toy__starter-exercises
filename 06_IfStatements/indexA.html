What do you call true and false values?

Booleans.

How is true similar or dissimilar from "true"? Can they be used in the same way?

The boolean true is part of a true/false (yes/no) pair, and "true" is just a string of the word true. They are not interchangable.

5 == '5' = true

5 == 'five' = false

5 == '6' = false

5 === '5' = false

5 == 2 + 3 = true

5 == 5 = true

5 === 5 = true
