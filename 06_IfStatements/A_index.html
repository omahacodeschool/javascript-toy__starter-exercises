How is true similar or dissimilar from "true"? Can they be used in the same way?

From research I have found that JS converts operands of differing type to then compare them. It seems that they are compared as each data type and all instances must evaluate to true to reach the end evaluation of true. In this case, when converting both to numbers, the Boolean is converted to '1' while "true" is converted to NaN. I wouldn't think that they could be used in the same way.
