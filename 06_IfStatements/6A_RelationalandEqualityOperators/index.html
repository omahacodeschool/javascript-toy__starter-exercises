What do you call true and false values?

Boolean

How is true similar or dissimilar from "true"? Can they be used in the same way?

true and "true" are dissimilar in the fact that true is a boolean value understood by the programming language whereas "true" is a string. Both of these values can be used for comparison purposes, however, the string value "true" would only be comparing the string and would produce a false value any time it is compared to anything other than itself. In this instance true is more beneficial since it can be used to compare the values of many other data types.
