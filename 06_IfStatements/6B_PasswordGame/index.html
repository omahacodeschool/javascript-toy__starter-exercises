Did you use = or == or === to compare passwords (or something else)? Why? Would any of the others work?

I used === in order to compare passwords. I used this because I read that === does not attempt to perform any conversion unlike ==. I believe that == would work, however, it may allow for small variations in the password which for all intents and purposes wouldn't be the best idea. I don't believe = would work at all because you would then be assigning the user input to you password which creates all sorts of issues. Not only would it always evaluate as true, but your password would also be at the mercy of your user being changed every time the program was run.

I would assume it's generally a bad idea since it would be easily accessible to anyone, especially in the event someone with malicious intent.
