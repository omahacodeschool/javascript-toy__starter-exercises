What do you call `true` and `false` values? How is `true` similar or dissimilar from `"true"`? Can they be used in the same way?

A true/false data type is a boolean. As I said above, the word "true" is not the same as the return value of true--you can write methods that return "true" if the value is true, or you can make it return "true" if the value is false if you REALLY want to mess with them.
