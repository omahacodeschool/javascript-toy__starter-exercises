2. What happens if the user clicks "OK" (or equivalent) without typing anything in? What information gets assigned to a variable, if any?

If the user clicks "OK" without typing a response, it appears that the value is saved as an empty string. That information still is assigned to the corresponding variable. This is different from the situation where you don't actually define a variable at all. You could do something like prompt("EXAMPLE QUESTION", "N/A") for a default string value of "N/A" if you didn't want to store an empty string but wanted a quick identifier to query by, in the event that people don't have to fill out certain forms.
