Given that you use either loop structure interchangeably, what do the different loop structures seem suited for?

The 'for' loop loops through a block of code a number of times. The 'while' loop loops through a block of code as long as a specified condition is true. The 'for' loop is executed before the code block starts, defines the condition for running the code block, and is executed each time after the code block has been executed. The 'while' loop is much the same as the 'for' loop except that it only defines the the condition for running the code block.
